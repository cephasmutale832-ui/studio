# **App Name**: AccessGate Pro

## Core Features:

- Document and Video Upload: Allows admins and authorized agents to upload videos and documents.
- Role-Based Access Control: Implements an admin role with full access and three agent roles with upload privileges.
- Student Data Capture: Captures and stores student details, including profile pictures.
- Access Restriction: Prevents access for students whose data is not captured or who haven't paid.
- Initial Access Code: Provides a temporary initial access code for accessing materials for the first seven days.
- Payment Validation Tool: Uses transaction details uploaded by the student during payment (Airtel, Apple statement, etc.) as a tool to automatically validate payment and grant extended access. If payment validation fails, the system automatically revokes access.

## Style Guidelines:

- Primary color: Deep indigo (#4B0082) for a secure and trustworthy feel.
- Background color: Light gray (#E6E6EA) to ensure readability and a clean interface in light mode.
- Accent color: Teal (#008080) for interactive elements and calls to action, providing a fresh and engaging contrast.
- Body font: 'Inter', a sans-serif typeface that will provide a clean, objective appearance.
- Headline font: 'Space Grotesk', a sans-serif typeface, suitable for short amounts of body text; if longer text is anticipated, use this for headlines and 'Inter' for body.
- Use clear, consistent icons for navigation and actions.
- Subtle transitions for loading states and form submissions.
